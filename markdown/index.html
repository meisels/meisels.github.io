Markdown to HTML Converter


So this got me thinking... how hard could it be? So I did a little Googling and copied the code from this site. Then I edited it a little to include more markdown stuff and added a GUI (Graphic User Interface). To see it, click here.

Just a fair warning, there are a few bugs. For example, the script is incapable of reading 2 sets of asterisks on 1 line. Use *1**2* (should produce 12 but instead produces 1**2).
