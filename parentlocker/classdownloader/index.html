I am lonely

That's why I made a NEW bookmarklet to download ParentLocker school lists.

See, there used to be a button that allowed you to download an excel spreadsheet of whatever class you were in.

Then, probably for security reasons, ParentLocker disabled this feature.

I knew the lists were still on the servers because my teachers still had the excel button.

So because I'm a lonely nerd, I built a short script that downloads it for you.

This project will be archived after June 2021 because I will not go to a school with ParentLocker anymore. If ParentLocker updates their security protocols, I won't even know about it.

BUGS:

Would you be surprised if I told you I found a bug?

2/8/21 - Click either of the PRINT buttons (you can click cancel afterwards) and then try to download the spreadsheet. The reason for this bug is because the print window adds a # to the URL after closing. The # is used for jumping to a certain element on the page by using its id. Because it is there, my script adds '&media=excel' after the # so it reads it as 'jump to the element with the id &media=excel'. So because there is none, it does nothing.
